# API Authentication Demo (Express + Axios + EJS)

This project demonstrates how to use different types of API authentication using Node.js, Express, Axios, and EJS.

## Features

- No Authentication request
- Basic Authentication
- API Key Authentication
- Bearer Token Authentication
- Server-side rendering using EJS

## Tech Stack

- Node.js
- Express.js
- Axios
- EJS


## Installation

1. Clone the repository

git clone https://github.com/your-username/api-auth-demo-express.git

2. Install dependencies

npm install

3. Run the server

node index.js

4. Open browser

http://localhost:3001

## Learning Objectives

- How to call APIs using Axios
- How to use different authentication methods
- How to render data using EJS
- How to build backend applications using Express
